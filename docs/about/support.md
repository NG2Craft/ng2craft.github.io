# Support
**Support is available on the public [L'escouade Discord](https://dsc.gg/lescouade-net)**

If you have any questions we can help you with join the Discord above, however if you have issues with
one of our Minecraft mods, create an Issue in the respective repository!

### Repositories
Use our main [Github Page](https://github.com/orgs/NG2Craft/repositories) to find the repository, or select
it from here:
- [Armor Profiles](https://github.com/NG2Craft/armor-profiles/issues/new)
- [Harder Progression](https://github.com/NG2Craft/harder-progression/issues/new)
- [Visible Armor Slots](https://github.com/NG2Craft/visible-armor-slots/issues/new)



Open an issue on the specific mod repository by filling out the given issue template on the respective repository. If you dont know how its done check the Documentation to use the proper Issue Format.

### Patience
If you have not gotten a response right away, **please be patient and stay connected**. Discord often is paced more like a forum than a chat room. Even though people are connected, they may actually be away from their computer when you join.
