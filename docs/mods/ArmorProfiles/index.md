# Armor Profiles

Create Armorsets and switch with custom Hotkeys for easier armor swapping, especially with mulitple mods.
The mod is easy to configure and allows a lot of customization.

Up to 3 set Armorslots that can be unlocked or hidden behind custom set milestones or achievements.
The mod is also compatible with the LevelZ mod and our upcoming Harder Progression aswell as the upcoming
Level System.
